# cnnp

Системы Поддержки Принятия Решений (и немного теории принятия решений)

Э-э-э значит, 5 работ

для работ 1, 3, 5 нужно применить npm install

**1. экспертная продукционная система**
Выбор наклеек кс2 по 3 группам параметров и предпочтениям пользователя. Цвет, размер, эффекты

**2. принятие решений в условиях неопределенности**
Оптимизация выбора станка для максимальной прибыли. условий мизер - вольности максимум. ещё там есть стратегии Грубый и Грубее

**3. Нейлоровская/Байесовская система**
Диагностика заболеваний желудка по вероятностям того или иного диагноза. Сумма всех вероятностей равна 1 - это важно, так как без ключевых симптомов результатом будет гастрит

**4. Теория игр**
2 производителя борются за рынок. Надо рассчитать в граничных условиях и в чистых стратегиях

**5. хз чёто-то про алгоритмы**
Даётся задача о ранце, и нужно её решить одним из методов: ветвей и границ, эволюционный, монте-карло, отжига. сравнение не забыть с полным перебором. Здесь выбран метод имитации отжига
