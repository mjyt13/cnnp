# cnnp
Системы Поддержки Принятия Решений (и немного теории принятия решений)

Э-э-э значит, 5 работ
**1. экспертная продукционная система**
Выбор наклеек кс2 по 3 группам параметров и предпочтениям пользователя. Цвет, размер, эффекты

**2. принятие решений в условиях неопределенности**
Оптимизация выбора станка для максимальной прибыли. условий мизер - вольности максимум

**3. Нейлоровская/Байесовская система**
Диагностика заболеваний желудка по вероятностям того или иного диагноза

**4. Теория игр**
2 производителя борются за рынок. Надо рассчитать в граничных условиях и на пох

**5. хз чёто-то про алгоритмы**
Даётся задача о коммивояжёре, и нужно её решить одним из методов: ветвей и границ, эволюционный, сравнение не забыть с полным перебором
